## 单点接地：模拟地、数字地与功率地如何协同

“地”不是一个绝对安静的零电位点，而是电流回到源端的路径。只要路径存在电阻、电感和寄生耦合，回流电流就会在地线上产生压降，使不同位置的“地”出现电位差。单点接地讨论的核心不是把地机械地分成几个名字，而是控制不同性质的回流电流：让敏感模拟电流、快速数字电流和大电流功率回流各走各的低阻路径，并在受控位置汇合。

在混合信号和电源电路中，常见地名有三类：

| 地名称      | 典型来源                             | 电流特点                          | 主要风险                  |
| -------- | -------------------------------- | ----------------------------- | --------------------- |
| AGND，模拟地 | [[ADC_Selection_and_Application|ADC]]、[[DAC_Selection_and_Application|DAC]]、[[Op_Amp_Classic_Circuits|运放]]、传感器、[[Voltage_Reference|电压基准源]] | 电流通常较小，但对噪声敏感 | 地电位波动直接变成测量误差、底噪或基准漂移 |
| DGND，数字地 | MCU、FPGA、逻辑接口、时钟、通信总线            | 边沿快，$di/dt$ 和 $dv/dt$ 大，高频分量多 | 地弹、串扰、EMI、采样噪声        |
| PGND，功率地 | [[Power_Management_IC|DC-DC]]、[[MOSFET_Selection_and_Switching|MOSFET]]、大电流负载、电机、继电器、LED 阵列 | 电流大，脉冲强，环路面积敏感 | 大电流压降、开关尖峰、地反弹、模拟参考污染 |

因此，单点接地不是“所有地只能接一个点”的教条，而是一个回流路径设计原则：**低噪声参考地不能承载大电流和高频开关回流；功率回流不能穿过模拟信号链；数字高速信号不能跨越地缝。**

### 为什么模拟地和数字地不能简单混在一起

理想导线没有阻抗，所有地都是 $0\ \mathrm{V}$。实际 PCB 上的地平面、走线、过孔、焊盘、连接器都有阻抗，可近似写成：

$$
Z_{\mathrm{GND}} = R + j\omega L
$$

当地电流流过这段阻抗时，会产生地噪声电压：

$$
V_{\mathrm{noise}} = I_{\mathrm{return}} Z_{\mathrm{GND}}
$$

其中 $I_{\mathrm{return}}$ 是回流电流。数字电路的电流边沿很快，例如 CMOS 输入电容充放电时有：

$$
i = C\frac{dv}{dt}
$$

边沿越陡，$\frac{dv}{dt}$ 越大，瞬态电流越大。这个电流必须经电源和地返回源端。如果数字回流穿过 ADC、运放、电压基准或传感器的参考地，数字噪声就会通过公共地阻抗叠加到模拟信号上。

| 模拟电路 | 被污染后的表现 |
|---|---|
| ADC 输入 | 采样码跳动、低位不稳定、有效位数下降 |
| DAC 输出 | 输出纹波、基准漂移、低频噪声升高 |
| 运放小信号放大 | 噪声被放大，出现偏置漂移或杂散波形 |
| 传感器接口 | 微弱信号被数字噪声淹没 |
| 电压基准 | 整个测量链路随基准地一起跳动 |
| 音频电路 | 底噪、啸叫、点击声或数字串音 |

所以，模拟地和数字地不能简单混在一起的根本原因不是“模拟”和“数字”这两个名字天然冲突，而是两类回流电流的幅值、频谱和敏感度不同。

### 为什么功率地也必须单独讨论

功率地 PGND 比数字地更容易污染系统地。数字电路的主要问题是边沿快，而功率电路同时具有大电流、快速开关和较大的寄生能量。典型位置包括 Buck/Boost 的开关管源极、电流采样电阻下端、输入输出电容负端、电机驱动桥臂、继电器线圈回路和大电流 LED 回路。

功率地的噪声主要来自两类机制。

#### 大电流产生的直流压降

即使只考虑铜箔电阻，功率回流也会产生电位差。这个问题在 [[Current_Sensing_and_Power_Measurement|电流检测]] 和毫欧级采样中尤其明显：

$$
\Delta V = I_{\mathrm{PGND}}R_{\mathrm{path}}
$$

例如负载电流为 $5\ \mathrm{A}$，某段地回路等效电阻为 $10\ \mathrm{m\Omega}$，这段地上的压降就是：

$$
\Delta V = 5\ \mathrm{A}\times 10\ \mathrm{m\Omega}=50\ \mathrm{mV}
$$

对于功率负载，$50\ \mathrm{mV}$ 也许不明显；但对于毫伏级传感器或 ADC 基准，这已经是很大的误差。

#### 快速开关产生的电感压降

功率开关回路中，寄生电感上的尖峰电压为：

$$
V_L = L\frac{di}{dt}
$$

当 MOSFET 或二极管电流快速变化时，即使只有几纳亨寄生电感，也可能产生明显尖峰。这个尖峰若通过地回路耦合到控制芯片或模拟采样地，会造成误触发、采样错误、环路振荡或 EMI 变差。

| 功率地噪声来源 | 常见位置 | 设计重点 |
|---|---|---|
| 负载大电流压降 | 电机、LED 阵列、加热片、电源输出端 | 加宽铜箔、缩短回路、避免与模拟地共用窄路径 |
| 开关电流尖峰 | DC-DC 开关管、续流二极管、输入电容回路 | 最小化热环路面积，输入电容靠近开关管 |
| 采样地偏移 | 电流采样电阻、反馈分压下端 | Kelvin 采样，信号地不要接在大电流焊盘之后 |
| 继电器/电机反灌 | 线圈、桥臂、电刷电机 | 续流、吸收、TVS、功率回路独立返回 |

### 分地的真正目标：控制回流路径

分地不是在原理图里画出 AGND、DGND、PGND 就结束。真正有效的设计要回答三个问题：

1. 每一股电流从哪里出来，又从哪里回去？
2. 它的回流路径是否穿过敏感参考点？
3. 它和其他信号共享的地阻抗是否足够小？

一个更准确的结构可以这样理解：

```text
传感器 / ADC / 运放 ── AGND ┐
MCU / 通信 / 时钟   ── DGND ├── 系统参考点 ── 电源入口 / 机壳 / 电池负端
DC-DC / MOS / 负载  ── PGND ┘
```

这个示意图只表达“受控汇合”的思想，并不表示所有 PCB 都必须用狭窄走线把三类地拉到一个星点。对于高速和高密度 PCB，完整连续地平面往往比割裂地平面更安全。布局上可以分区，铜皮上未必一定割裂。

### 单点接地、完整地平面与割地的边界

很多初学设计会把“单点接地”理解成“必须切开地平面，只留一个细桥连接”。这在低频、小系统或大电流汇流场景中有时有效，但在高速数字和混合信号 PCB 中可能适得其反。

| 做法 | 适用条件 | 风险 |
|---|---|---|
| 完整地平面 + 分区布局 | 大多数 MCU、ADC、DC-DC 混合板 | 需要靠布局约束回流，不能让功率路径穿过模拟区 |
| AGND/DGND 单点连接 | 低速混合信号、ADC 数据手册明确推荐、板上噪声路径清楚 | 连接点选错会迫使回流绕路 |
| PGND 与 SGND 单点连接 | 开关电源、功率驱动、采样控制分区明确 | 采样地若接到大电流路径末端，会把压降带入反馈 |
| 大面积割地 | 低频隔离、安规隔离、特殊噪声约束 | 高速信号跨地缝会导致回流绕行和 EMI |
| 多点随意连接 | 试图降低阻抗但缺少路径规划 | 形成不可控环流，噪声定位困难 |

> [!warning] 不要让高速信号跨越地缝
> 高速信号的回流电流倾向于沿信号线正下方返回。如果信号跨越地平面裂缝，回流路径会被迫绕远，环路面积变大，反射、串扰和辐射都会恶化。阻抗连续性和端接判断可结合 [[Impedance_Matching|阻抗匹配]] 一起检查。

### 功率地与信号地如何连接

在电源芯片和功率驱动中，常见名称还有 SGND、AGND、GND、PGND。SGND 通常指控制芯片、反馈分压、补偿网络、电流采样信号使用的安静地；PGND 指开关管、电感电流、输入输出电容和负载大电流回流使用的功率地。

#### DC-DC 中的典型连接原则

1. 输入电容负端、功率 MOSFET、续流器件和 PGND 引脚构成高 $di/dt$ 热环路，应尽量短、宽、紧凑。
2. 反馈分压下端、补偿网络、电压基准旁路应接到 SGND，避免接在大电流 PGND 路径上。
3. SGND 与 PGND 通常在芯片推荐点、采样电阻 Kelvin 点或输出电容负端附近单点汇合。
4. 如果芯片同时给出 AGND/SGND 与 PGND 引脚，应优先按数据手册的布局示例连接，不要自行把它们远距离短接。

#### 电流采样中的 Kelvin 连接

电流采样电阻同时属于功率路径和信号测量对象。若采样线直接接在大电流铜箔上，测到的不只是电阻压降，还包括铜箔和焊点压降。Kelvin 采样用两根独立细线从采样电阻内侧引出信号，避免大电流路径压降进入测量结果。

```text
错误：负载电流与采样信号共用一段铜箔

电源 ── 铜箔 ── Rsense ── 铜箔 ── 负载
             ↑        ↑
          采样+     采样-

正确：采样线从电阻焊盘内侧单独引出

电源 ── 铜箔 ── Rsense ── 铜箔 ── 负载
              │      │
            Sense+  Sense-
```

这里的重点不是“地名”，而是采样回路不能承载负载电流。

### 模拟地、数字地、功率地连接方式对比

| 连接方式   | 等效特性              | 适用情况                                       | 优点                       | 使用注意                           |
| ------ | ----------------- | ------------------------------------------ | ------------------------ | ------------------------------ |
| 直接铜皮连接 | 最低阻抗，宽频连接         | 地平面完整、布局分区清楚、回流路径明确；量产验证后固定方案              | 阻抗最低，可靠性最好，不引入额外器件寄生     | 不能让数字大电流或功率回流穿过模拟参考区           |
| 0 欧姆电阻 | 直流低阻，可装可不装        | AGND/DGND 或 SGND/PGND 调试预留；需要比较铜皮、磁珠、断开等方案 | 成本低，便于调试和版本配置，可替换为磁珠或小电阻 | 不是理想导线；大电流要看额定电流；高频下有封装电感      |
| 磁珠     | 低频近似导通，高频阻抗较大且有损耗 | 高频噪声频段明确；模拟电源与数字电源隔离；厂家推荐的局部隔离             | 可衰减特定频段噪声，常用于电源支路隔离      | 可能与电容谐振；直流偏置会降低阻抗；不宜盲目放在主地回流路径 |
| 电容     | 隔直通交，只提供交流回流      | 两侧需要直流隔离但要提供高频参考；EMI 共模回流；隔离系统跨隔离带         | 给高频噪声提供受控回路              | 不能提供直流参考；可能形成浮地；安规位置必须使用 Y 电容  |
| 电感     | 低频导通，高频阻抗升高       | 电源支路 LC 滤波、低频能量隔离                          | 可限制电流快速变化                | 容易饱和和谐振；寄生参数明显；一般不作为地连接首选      |
| 小电阻    | 有意引入阻尼或限流         | 特殊测量、抑制地环路、小电流参考连接                         | 可提供阻尼，降低某些谐振尖峰           | 会产生地电位差，不适合承载大回流电流             |
| 导线/跳线  | 低频低阻，布线灵活         | 原型调试、机箱接地、板间连接                             | 易改动，便于验证                 | 高频下电感很大；长导线会形成环路天线             |

### 什么时候用 0 欧姆电阻

0 欧姆电阻适合“需要明确直流连接，同时希望保留调试余地”的位置。它常用于 AGND 与 DGND 的连接点、SGND 与 PGND 的连接点、板级地策略 A/B 测试，以及同一 PCB 的多版本配置。

适用场景：

| 场景 | 说明 |
|---|---|
| 新板验证 | 先用 0 欧姆短接，若噪声不理想可替换为磁珠、小电阻或断开 |
| ADC/DAC 地汇合 | 在芯片推荐连接点附近作为可调试单点 |
| DC-DC SGND/PGND 汇合 | 作为信号地和功率地的受控连接点 |
| 生产版本配置 | 不同版本通过贴装状态选择地连接方案 |
| 电流路径排查 | 拆除后可测量某条回流路径对噪声的影响 |

使用边界：

| 注意项 | 原因 |
|---|---|
| 不能承载未核算的大电流 | 0 欧姆电阻有额定电流和最大阻值 |
| 不能修复错误布局 | 若功率回流本来穿过模拟区，0 欧姆只是把问题集中到一点 |
| 高频下不是短路 | 焊盘和封装会引入寄生电感 |
| 放置位置要靠近汇合点 | 远离关键芯片会拉长敏感回流路径 |

### 什么时候用磁珠

磁珠适合处理频段相对明确的高频噪声。它在低频下阻抗较小，在高频下阻抗增大，并通过损耗把高频能量转化为热。磁珠更常用于电源支路隔离，而不是随意串在主地线上。

常见可靠用法是数字电源先经磁珠进入模拟电源，再在模拟侧就近去耦：

```text
数字电源 ── 磁珠 ── 模拟电源
                  │
                去耦电容
                  │
                 AGND
```

磁珠选型应关注：

| 参数 | 用途 |
|---|---|
| 阻抗频率曲线 | 判断目标噪声频段是否落在有效阻抗区 |
| 额定电流 | 防止直流偏置过大使磁珠阻抗下降 |
| 直流电阻 DCR | 评估压降和发热 |
| 阻抗组成 | 电阻性损耗越高，越有利于吸收噪声；纯电感性更容易谐振 |
| 与电容组合 | 检查是否形成尖锐 LC 谐振 |

磁珠不是“高频垃圾桶”。如果噪声频率、回流路径和负载电流都不清楚，盲目加磁珠可能无效，甚至放大某些频点。

### 什么时候用电容

电容用于提供高频交流回流路径，但不能提供直流参考。它适合直流需要隔离、高频又不能完全悬浮的场景。

| 场景 | 连接目的 |
|---|---|
| 隔离电源两侧 | 降低高频共模电位差 |
| 机壳地与信号地 | 给 ESD/EMI 高频电流提供路径 |
| 安规隔离边界 | 使用 Y 电容提供受控高频耦合 |
| 高频旁路 | 让高频噪声就近闭合，不穿过敏感路径 |

使用电容连接两块地时要注意：低频和直流下两侧仍可能浮动；普通电容不能跨越安全隔离边界；电容的 ESL、ESR 和自谐振频率会限制高频效果。电容值也不是越大越好，过大的电容可能改变系统漏电流、启动行为或 EMI 频谱。

### 什么时候用电感

电感主要用于电源滤波和支路隔离，不宜作为 AGND/DGND 或 SGND/PGND 的默认连接方式。原因是地连接需要稳定参考，而电感会随频率升高引入阻抗，还可能饱和或与电容形成谐振。

适用场景：

| 场景 | 说明 |
|---|---|
| LC 电源滤波 | 降低某个电源支路的纹波 |
| 低频能量隔离 | 限制某支路电流变化 |
| 已知频段抑制 | 噪声频率明确且可验证谐振风险 |

如果把电感串在地线上，应非常谨慎。当地线上电流变化时，电感两端会产生电压差，反而让两个区域的参考电位不一致。电感与磁珠的功能边界可参考 [[Passive_Components-L|认识无源器件：电感]]。

### 什么时候直接铜皮连接

直接铜皮连接是最可靠、最低阻抗的连接方式，适合布局已经控制好回流路径的场景。很多现代混合信号 PCB 并不需要把地平面切碎，而是使用完整地平面，通过器件摆放和走线方向让模拟、数字和功率回流自然分区。

适用条件：

| 条件 | 说明 |
|---|---|
| 模拟、数字、功率器件分区清楚 | 各自电流回路就近闭合 |
| 高速信号不跨越地缝 | 回流路径连续 |
| 功率热环路局部最小 | DC-DC 输入电容、开关管和 PGND 紧凑 |
| 模拟参考远离大电流路径 | ADC、运放、基准不承载功率回流 |
| 已通过实测验证 | 噪声、纹波、EMI 和温升符合要求 |

### 实际设计流程

#### Step1: 画电流回路，而不是先命名地

在原理图上标出以下回路：

| 回路 | 需要检查 |
|---|---|
| 模拟输入回路 | 传感器信号和参考地是否同路径返回 |
| ADC/DAC 参考回路 | 基准、输入和芯片地是否靠近 |
| 数字时钟/通信回路 | 高速信号下方是否有连续地参考 |
| DC-DC 热环路 | 输入电容、开关管、续流路径是否最小 |
| 负载大电流回路 | 是否穿过模拟地或控制地 |
| 机壳/接口 ESD 回路 | 静电是否有短路径泄放到机壳或入口地 |

#### Step2: 决定连接策略

| 判断条件 | 推荐策略 |
|---|---|
| 低速、小电流、噪声不敏感 | 完整地平面，少分割 |
| ADC/运放很敏感，数字电路不复杂 | AGND/DGND 在 ADC 附近或数据手册推荐点汇合 |
| 存在 DC-DC 或大电流负载 | PGND 独立形成最小回路，SGND/AGND 在安静点与 PGND 汇合 |
| 需要调试不同连接方式 | 用 0 欧姆电阻预留 |
| 高频噪声已定位在某电源支路 | 优先在电源支路使用磁珠加去耦 |
| 存在隔离边界或机壳地 | 按安规和 EMC 目标选择电容、Y 电容、RC 或放电器件 |

#### Step3: 测试验证

地策略不能只靠规则判断，应结合实测验证：

| 测试项 | 观察目标 |
|---|---|
| ADC 码流噪声 | 数字开关、电源负载变化时是否跳动 |
| 电源纹波 | PGND 尖峰是否耦合到模拟电源 |
| 示波器地弹测量 | 大电流开关时各地点电位差 |
| 近场扫描 | 高频噪声是否集中在地缝、热环路或连接器附近 |
| EMI 测试 | 接地方式是否改变辐射或传导峰值 |
| 温升测试 | 大电流地路径是否过热 |

### 常见错误

| 错误做法 | 问题 |
|---|---|
| 看到 AGND/DGND 就切地 | 可能破坏高速信号回流 |
| 把 PGND 和 AGND 在远处相连 | 大电流压降会抬高模拟参考地 |
| 反馈分压下端接到功率地末端 | 输出电压采样会包含功率地压降 |
| 让采样线与负载电流共用铜箔 | 测到铜箔压降而不是器件真实压降 |
| 用磁珠随便串地 | 可能引入谐振和不可控参考电位 |
| 电容跨接两地但没有直流参考 | 低频下可能浮地 |
| 0 欧姆电阻承载大电流但不查额定值 | 会产生压降、发热甚至失效 |
| 高速信号跨越地缝 | 回流绕路，EMI 和串扰变差 |

### 选型和布局小结

| 设计对象 | 首要目标 | 推荐做法 |
|---|---|---|
| 模拟地 AGND | 保持参考点安静 | 靠近模拟器件闭合回流，远离 PGND 大电流 |
| 数字地 DGND | 保持高速回流连续 | 完整参考平面，不跨地缝 |
| 功率地 PGND | 承载大电流和开关尖峰 | 最小热环路，宽铜皮，多过孔，避开模拟参考 |
| SGND/PGND 连接 | 让控制参考不被功率电流污染 | 在芯片推荐点、采样 Kelvin 点或输出电容安静端汇合 |
| AGND/DGND 连接 | 统一系统参考，同时避免数字回流污染模拟区 | 优先按芯片手册；新板可用 0 欧姆预留 |
| 机壳地/保护地 | 泄放 ESD 和共模噪声 | 接口入口短路径连接，必要时用电容、RC 或浪涌器件 |

一句话总结：单点接地真正要解决的是公共阻抗耦合和回流路径失控。模拟地怕噪声，数字地怕回流不连续，功率地怕大电流和开关尖峰；三者最终可以连接，但连接点必须服务于电流路径，而不是服务于符号名称。
