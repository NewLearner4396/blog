## MOS 管基础与常见应用：辨别、驱动、电源切换与布局

MOS 管通常指 MOSFET，即金属氧化物半导体场效应管。它最常见的用途是开关、电平转换、电源路径控制、理想二极管、防倒灌、马达/继电器驱动和功率变换。本文重点讲基础辨别、JFET 与 MOSFET 的区别、常见驱动电路和 PCB 使用注意点；具体损耗、SOA、开关速度和功率选型见 [[MOSFET_Selection_and_Switching|MOSFET 选型与开关应用]]。

### MOSFET 的三个端子

MOSFET 的三个基本端子是：

- G：Gate，栅极，控制端。
- D：Drain，漏极，主电流端之一。
- S：Source，源极，主电流端之一。

增强型 MOSFET 的栅极与沟道之间由氧化层绝缘，因此直流栅极电流极小。但栅极不是“完全不用驱动”，它等效为电容，开关时需要充放电。

### N 沟道和 P 沟道

| 类型 | 导通条件 | 常见用途 | 直觉 |
|---|---|---|---|
| N-MOS | $V_{GS}$ 足够为正 | 低边开关、同步整流、高效功率开关 | 导通电阻低、选择多 |
| P-MOS | $V_{GS}$ 足够为负 | 简单高边开关、电源路径 | 驱动简单但导通电阻通常更高 |

注意 $V_{GS}$ 是栅极相对源极的电压，不是栅极相对地的电压。高边 N-MOS 的源极会跟随输出升高，所以栅极必须被驱动到高于源极的电压，常用自举或隔离驱动。

### 如何辨别 MOS 管

#### 看丝印和数据手册

最可靠的方法是查丝印、封装和数据手册。SOT-23、SOT-223、TO-220、DFN 等封装的引脚排列可能不同，同样叫 G/D/S，位置不一定一致。

#### 看体二极管方向

功率 MOSFET 通常有体二极管。对 N-MOS，体二极管一般从源极指向漏极；对 P-MOS，方向相反。用万用表二极管档可粗略判断 D/S 方向，但不能替代查数据手册。

```text
N-MOS 低边开关常见方向：

Load ── Drain
          |
        MOSFET
          |
        Source ── GND

体二极管方向：Source -> Drain
```

#### 不要把阈值电压当作驱动电压

$V_{th}$ 只是 MOSFET 在很小测试电流下刚开始导通的栅源电压，不表示能承载负载电流。实际选型要看目标 $V_{GS}$ 下的 $R_{ds(on)}$，例如 $V_{GS}=2.5\ \mathrm{V}$、4.5 V、10 V。

### JFET 与 MOSFET 的区别

| 项目 | JFET | MOSFET |
|---|---|---|
| 栅极结构 | PN 结栅极 | 绝缘氧化层栅极 |
| 输入电流 | 反偏 PN 结漏电 | 极小直流漏电，但有栅电容 |
| 常见状态 | 多为耗尽型，默认导通 | 增强型最常见，默认关断 |
| 典型用途 | 低噪声高阻输入、恒流二极管、模拟开关 | 功率开关、电平转换、电源路径 |
| 脆弱点 | 栅结过压 | 栅氧 ESD/过压 |

JFET 更像一个用反偏结控制的电阻，MOSFET 更适合用绝缘栅控制大电流开关。两者都属于场效应管，但驱动和应用边界不同。

### 低边 N-MOS 开关

```text
VCC ── Load ── Drain
                │
              N-MOS
                │
Source ─────────┴── GND

MCU GPIO ── Rg ── Gate
              │
             Rpd
              │
             GND
```

低边 N-MOS 是最常用开关。优点是驱动简单，MCU 直接给栅极高电平即可导通。关键设计点：

- 选择逻辑电平 MOSFET，确认实际 GPIO 电压下 $R_{ds(on)}$ 足够低。
- 栅极串联 $R_g$ 控制开关速度和振铃。
- 栅极下拉 $R_{pd}$ 防止上电悬空误导通。
- 感性负载必须有续流、TVS 或吸收路径。
- 负载地被开关，不适合地参考敏感负载。

### 简单 P-MOS 高边开关

```text
VCC ── Source
        │
      P-MOS
        │
Drain ── Load ── GND

Gate 上拉到 Source -> 关断
Gate 拉低          -> 导通
```

P-MOS 高边开关可以保持负载地不被抬高，适合小电流电源开关。缺点是 P-MOS 同尺寸导通电阻通常高于 N-MOS，大电流效率较差。栅源电压不能超过最大额定值，较高输入电压时要加限压。

### 高边 N-MOS 开关

高边 N-MOS 效率高，但驱动复杂。因为导通后源极接近电源电压，栅极必须高于源极：

$$
V_G=V_S+V_{GS,on}.
$$

常用方案：

- 自举高边驱动：适合半桥、同步 Buck 等有周期性开关的场景。
- 电荷泵高边驱动：适合静态高边负载开关。
- 隔离驱动：适合高压或隔离安全场景。
- 专用 load switch/eFuse：适合电源路径、限流、软启动和保护。

### 半桥和电机驱动

```text
       VBUS
        │
      High-side MOS
        │──── Switch node ─── Motor/inductor
      Low-side MOS
        │
       GND
```

半桥驱动要重点关注：

- 高低边不能同时导通，需要 dead time。
- 高边自举电容要满足占空比和栅电荷。
- 开关节点 dv/dt 会通过米勒电容误导通低边 MOS。
- 源极公共电感会造成地弹和栅源负压。
- 功率环路面积决定尖峰、EMI 和损耗。

### 电平转换

小信号 N-MOS 常用于 I2C 双向电平转换：

```text
3.3V pullup ── SDA_L ──┬── D/S MOS S/D ── SDA_H ── 5V pullup
                       │
                      Gate -> 低压侧电源
```

这种电路适合开漏总线，不适合高速推挽信号。高速接口应使用专用电平转换器。

### 理想二极管和防倒灌

普通二极管有正向压降，功耗为：

$$
P\approx V_F I.
$$

MOSFET 理想二极管方案通过控制 MOSFET 导通方向，用很低的 $R_{ds(on)}$ 替代二极管压降：

$$
P\approx I^2R_{ds(on)}.
$$

背靠背 MOSFET 可以阻断双向体二极管路径，常用于防倒灌、电源 ORing、电池保护和热插拔。

```text
背靠背 MOS：

Input ── MOS ──┤├── MOS ── Output
       体二极管相反，关断时双向阻断
```

### 主副电源自动切换

主副电源切换常见目标是：主电源存在时用主电源，主电源掉电时自动切到电池或备用电源，同时避免反灌。

简单二极管 ORing 成本低，但有压降。MOS 理想二极管或专用 power mux 可以降低压降，并提供限流、软启动、优先级和保护。若系统电源路径复杂，优先考虑专用电源路径管理芯片，而不是用多个 MOS 临时拼。

### 栅极驱动细节

MOSFET 栅极等效为电容，驱动速度近似由栅电荷和驱动电流决定：

$$
t_{sw}\approx \frac{Q_g}{I_{gate}}.
$$

栅极电阻不是固定值越小越好。它用于控制开关速度、抑制振铃、降低 EMI 和限制驱动峰值电流。需要更快关断时，可用二极管并联栅极电阻，实现开通和关断不同阻值。

```text
driver ── Ron ──┬── Gate
        ──|<|── Roff ┘
```

### PCB 布局注意点

#### 功率回路最小化

开关电流回路面积越大，寄生电感越大，尖峰越严重：

$$
V=L\frac{\mathrm{d}i}{\mathrm{d}t}.
$$

输入电容、MOSFET、负载或电感要形成最小热环路。

#### 栅极回路短

栅极驱动线长会振铃。驱动器应靠近 MOSFET，栅极电阻靠近 Gate 放置，Gate 和 Source 回路紧耦合。

#### 使用 Kelvin Source

大电流源极压降会叠加到栅源驱动上，导致误导通或开关速度不可控。带 Kelvin Source 的封装应把驱动源极和功率源极分开连接。

#### 热设计

小封装 MOSFET 的电流能力常被 PCB 铜皮限制。数据手册最大电流通常基于理想散热条件，实际应按热阻和板级铜面积计算。

| 器件/系列 | 类型 | 常见用途 | 主要注意点 |
|---|---|---|---|
| 2N7002 | 小信号 N-MOS | 电平转换、小信号开关 | 电流能力小 |
| BSS138 | 小信号 N-MOS | I2C 电平转换 | 不适合大电流 |
| AO3400/AO3400A | 小封装 N-MOS | 低压小负载开关 | 热能力受 SOT-23 限制 |

### 常见错误

| 错误 | 后果 |
|---|---|
| 用 $V_{th}$ 判断能否驱动负载 | MOS 半导通、发热烧毁 |
| 栅极悬空 | 上电误导通 |
| 感性负载没有续流路径 | 关断尖峰击穿 MOS |
| 高边 N-MOS 直接用 MCU 拉高 | $V_{GS}$ 不够，不能真正导通 |
| 忽略体二极管 | 关断时仍有反向电流路径 |
| 栅极线又长又细 | 振铃、EMI、误导通 |

### 小结

MOSFET 应用的基本判断链是：先分清 N/P 沟道和体二极管方向，再确认 $V_{GS}$ 是否足够，随后按电流、热、驱动、保护和布局闭合设计。小信号 MOS 关注漏电、电容和电平，功率 MOS 关注 $R_{ds(on)}$、栅电荷、SOA、热和寄生参数。
